struct json_object *safuJsonAddNewArray(struct json_object *jobj, const char *name)¶
[source]

Add an empty json array value to the given json object. This function is mostly a wrapper for json-c’s json_object_new_array

Parameters:

  • jobj (json_object*) – Json Object to which an array should be added.

  • name (const char*) – Key to store the new array under.

Returns:

  • NULL in error case

  • Pointer to json array

struct json_object *safuJsonAddNewTimestamp(struct json_object *jobj, const char *name, const struct timespec *timestamp)¶
[source]

Add Timestamp to an json object. A Timestamp is a json array containing two json int64 numbers, which encode the seconds and nanoseconds of a unix time stamp.

Parameters:

  • jobj (json_object*) – The json object, who’s timestamp is to be updated

  • name (const char*) – Key the time stamp is stored under.

  • timestamp (const struct timespec*) – The timespec data to be encoded and stored.

Returns:

  • NULL in Error cases

  • Pointer to the json array containing the timestamp.
